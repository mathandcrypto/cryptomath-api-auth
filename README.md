# cryptomath-api-auth
**Auth** microservice for managing authentication sessions.

## Installation

```bash
$ npm install
```

## Prisma migrate

```bash
# development
$ npm run migrate:dev

# production mode
$ npm run migrate:deploy
```

## Generate prisma client

```bash
$ npm run prisma:generate
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
